# Agent Recommendations

This file contains recommendations and instructions for AI agents working on this codebase.

## Linting
- **Always run the linter** before finishing a task to ensure code quality and consistency.
- Command: `npm run lint`

## Internationalization (i18n)
- This project uses `react-i18next` for translations.
- **Add translations** for any new text or components added during a task.
- Translation files are located in `src/locales/`:
  - English: `src/locales/en.json`
  - Spanish: `src/locales/es.json`
- Ensure that both files are updated to maintain parity between languages.
