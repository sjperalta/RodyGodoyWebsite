# Arquitecto Rody Godoy - Website

Sitio web moderno, responsivo y profesional para el estudio de arquitectura **Arquitecto Rody Godoy**, basado en Puerto Cortés, Honduras.

## 🚀 Tecnologías Utilizadas

- **React** (Vite)
- **Tailwind CSS** (Estilizado)
- **Framer Motion** (Animaciones)
- **Lucide React** (Iconografía)

## 🛠️ Instalación y Desarrollo

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Ejecutar servidor de desarrollo:**
   ```bash
   npm run dev
   ```

3. **Construir para producción:**
   ```bash
   npm run build
   ```

## 📦 Despliegue en GitHub Pages

Este proyecto está configurado para ser desplegado fácilmente en GitHub Pages.

1. Asegúrate de que el campo `base` en `vite.config.js` esté configurado como `./` o el nombre de tu repositorio.
2. Ejecuta el comando de despliegue:
   ```bash
   npm run deploy
   ```
   *Esto construirá el proyecto y subirá el contenido de la carpeta `dist` a la rama `gh-pages`.*

## 📂 Estructura del Proyecto

- `src/components/`: Componentes modulares (Navbar, Hero, Services, etc.)
- `src/styles/`: Estilos globales y configuración de Tailwind.
- `src/assets/`: Imágenes y recursos estáticos.

## ✒️ Créditos

- **Diseño y Desarrollo:** IA Assistant (Antigravity)
- **Imágenes:** Placeholder images de [Unsplash](https://unsplash.com/). Reemplazar con fotos reales de proyectos en `src/components`.
